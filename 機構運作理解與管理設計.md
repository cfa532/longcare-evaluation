# 居家長照機構運作理解與管理設計

**機構名稱**：專安誠健康事業有限公司附設高雄市私立專安誠居家長照機構  
**文件性質**：運作理解記錄 ＋ 表單與 Agent 設計依據  
**撰寫日期**：2026-06-13  
**依據**：115年度居家式長期照顧服務機構評鑑基準、實際運作訪談

---

## 一、組織架構

```
總經理（業務負責人）
        │
        ├── 督導 A（Director A）
        │       ├── 照服員 1
        │       ├── 照服員 2
        │       └── 照服員 N
        │
        └── 督導 B（Director B）  ← 視規模可能僅1位督導
                ├── 照服員 1
                └── 照服員 M
```

**現況規模**：1–2 個督導團隊，每隊管理數名至十數名照服員。  
**評鑑法規要求**：每位督導最多服務 60 名個案（A6-2），督導應為**專任**（A6-3）。

## 二、各層級角色與職責

### 2.1 總經理（業務負責人）

**身份地位**
- 評鑑時代表機構出席（缺席則 A1/A2/A3/A9/B1/B8/D1 七項逕評 E）
- 向董監事會報告，並取得年度業務計畫通過（A1-4）

**核心職責**
| 職責 | 頻率 | 對應評鑑指標 |
|---|---|---|
| 審閱督導月報，掌握各團隊運作 | 每月 | A3 |
| 召開行政聯繫會議（含服務品質/工作改善） | 每季（至少） | A3-1、A3-2 |
| 訂定年度業務計畫，追蹤執行績效 | 每年 | A1 |
| 辦理服務滿意度調查，提送會議討論 | 每年 | D4 |
| 確保全員勞/健保、勞退提繳完整 | 持續 | A5-3 |
| 評鑑前準備佐證文件、接待評鑑委員 | 評鑑年 | 全部 |

**總經理不需要**：每日逐筆確認打卡、直接填寫個案記錄。這些由督導負責。

---

### 2.2 督導（Director）

督導是日常運作的**核心角色**，直接掌管一個服務團隊。

**每日職責**
- 確認所轄照服員出勤狀況（到班/請假/代班）
- 處理緊急狀況（個案臨時需求、照服員無法到班）
- 收取照服員繳回的紙本服務紀錄單

**每月職責**
| 職責 | 工具 | 對應指標 |
|---|---|---|
| 彙整照服員紙本紀錄 → 輸入督導月報表 | 督導月報.xlsx | A9、B5 |
| 對每位個案至少電訪一次 | 督導月報.xlsx（電訪欄） | B7-2 |
| 處理個案申訴/意見反應，填寫記錄 | 日常作業記錄簿 → 意見反應申訴記錄 | D3 |
| 開立個案服務收據，存根入帳 | 機構管理台帳 → 收費標準紀錄 | D2 |

**每季職責**
| 職責 | 工具 | 對應指標 |
|---|---|---|
| 對每位個案至少到府家訪一次 | 日常作業記錄簿 → 督導訪視紀錄 | B5、B7-2 |
| 重新確認個案需求評估（或確認無變化） | 個案照護管理 → 需求評估紀錄 | B2-3 |
| 追蹤個案照顧計畫執行成效 | 個案照護管理 → 照顧計畫追蹤評值 | B4 |
| 辦理跨專業個案討論會（≥3領域參與） | 個案照護管理 → 跨專業轉介個案討論 | B6-2 |
| 對所轄照服員執行個別督導並記錄 | 機構管理台帳 → 行政聯繫督導記錄 | A3-3 |

**新個案開案時（7工作天內完成）**
1. 初訪家庭，完成需求評估
2. 擬定照顧計畫，向個案/家屬說明
3. 指派照服員，說明服務時間/頻率
4. 建立個案資料至「個案開結案管理」

**督導不需要**：填寫複雜的統計報表。只需填寫紙本 → 抄入 Excel 月報 → 交給 GM。

---

### 2.3 照服員（Caregiver）

照服員是第一線服務提供者，教育程度不一，應**最小化文書負擔**。

**每次服務**
- 到府後填寫紙本「服務紀錄單」（打勾 + 簽名，約 2 分鐘）
- 服務結束後交給個案/家屬確認簽名
- 月底繳回督導彙整

**服務紀錄單設計原則**（見第四章）
- 全部打勾，不需寫長句
- 服務項目直接列出，照服員勾選完成項目
- 唯一需要手寫：時間（到/離）、異常簡述（一行即可）
- A4 單面，一張對應一次服務

**照服員不需要**：填寫評估表、照顧計畫、督導記錄。這些全由督導填寫。

## 三、日常作業流程（資料流）

### 3.1 每日流程

```
照服員到府 → 填紙本服務紀錄單（打勾）→ 個案/家屬確認簽名
     ↓
督導收取紀錄單，確認出勤正常
     ↓
有異常 → 督導立即處理，填「緊急異常事件報告」→ 回報 GM
```

### 3.2 每月流程

```
月底前
照服員繳回全月紙本紀錄單
     ↓
督導批次彙整 → 輸入「督導月報表」（約30分鐘/月）
  ・出勤統計
  ・服務次數統計
  ・月電訪記錄
  ・本月異常事項
     ↓
督導月報 → 送交 GM 審閱
     ↓
GM 簽核存檔 → 成為評鑑佐證
```

### 3.3 每季流程

```
督導執行：
  ・個案家訪（督導訪視紀錄）
  ・個案需求重評估
  ・跨專業個案討論會（B6，需≥3領域）
  ・照服員個別督導
     ↓
GM 召開行政聯繫會議（含服務品質/改善議題）→ 會議紀錄存檔
```

### 3.4 每年評鑑準備流程

```
全年累積的記錄（督導月報×12、訪視紀錄、照顧計畫…）
     ↓
GM 或指定行政人員整理佐證文件
     ↓
115年7月6日前送達「基本資料表」及「自評表」
     ↓
評鑑委員到訪（業務負責人必須出席）
  ・文件查閱
  ・現場訪談（GM + 1位陪評照服員）
  ・個案抽訪
```

## 四、表單設計原則（分層設計）

### 原則一：依使用者設計，而非依評鑑指標設計

| 使用者 | 設計方向 | 完成時間目標 |
|---|---|---|
| 照服員 | 打勾為主，零文字，A4 單面 | 每次服務 < 3 分鐘 |
| 督導 | 批次輸入，一表涵蓋多項指標 | 每月彙整 < 1 小時 |
| GM | 只看彙整結果，簽核即可 | 每月 < 15 分鐘 |

### 原則二：一張表多用途

督導每月填寫一張「督導月報表」，自動對應多個評鑑指標：

| 月報欄位 | 對應評鑑指標 |
|---|---|
| 出勤/服務次數統計 | A9（派案排班）、B5（服務查核） |
| 月電訪記錄欄 | B7-2（每月電訪） |
| 本月異常事項 | B8（緊急事件）、D3（申訴） |
| 新開案/結案記錄 | B7-1（開結案管理） |
| 月底服務件數 | D2（收費依據） |

### 原則三：紙本 ↔ 電子對應

| 場景 | 使用紙本 | 對應電子記錄 |
|---|---|---|
| 照服員每次服務 | 服務紀錄單（打勾+簽名） | 督導月底彙整入 Excel |
| 督導家訪/初訪 | 督導訪視紀錄（現場填） | 拍照存入電子版本 |
| 緊急事件 | 現場先記紙本 | 24小時內補填電子版 |

### 原則四：紙本表單設計規範（照服員用）

1. **大字體**（≥12pt），避免小字
2. **全打勾**，僅時間需手寫
3. **服務代碼對應中文**（不寫 BA07，寫「協助沐浴洗頭」）
4. **異常欄位放最後**，正常時留空即可
5. **一張紙 = 一次服務** = 一個個案

## 五、Agent 角色重新對應（對應真實職位）

原設計的 4 個 Agent 重新對應實際人員：

| Agent | 實際對應人員 | 主要任務 |
|---|---|---|
| **Agent A（個案照護督導員）** | 各督導（Director A / B） | 個案日常管理、月報彙整、電訪家訪、照顧計畫 |
| **Agent B（人員訓練管理員）** | GM 兼任 或 指定行政 | 員工花名冊、健檢、職前訓練、在職教育積分 |
| **Agent C（機構行政品質稽核）** | GM | 行政聯繫會議、年度計畫、滿意度調查、申訴彙整 |
| **Agent D（評鑑準備總管）** | GM（評鑑年） | 整合所有佐證，準備評鑑文件，應答評鑑委員 |

**小型機構的簡化原則**：Agent B/C/D 職責可合併由 GM 統籌，依工作時程分配即可。

## 六、督導月報表設計（核心工具）

督導月報表是整套系統的**樞紐文件**，月底 30 分鐘完成，成為大多數評鑑佐證的來源。

### 月報表結構

**第一頁：出勤與服務統計**
- 每位照服員：出勤天數、服務個案數、服務總時數
- 異常記錄：無故缺席/代班/調班（A9）

**第二頁：個案狀況彙整**
每個個案一行：
- 本月服務次數、電訪完成（B7-2）、狀態（正常/待處理/結案）
- 備註：異常事項、計畫修正需求、轉介需求

**第三頁：本月事件記錄**
- 緊急/意外事件（B8）
- 申訴/意見反應（D3）
- 缺失改善追蹤更新（A4）

**督導簽名 → 送 GM 簽核 → 歸檔**

## 九、立即需要建立的記錄習慣

以下是**評鑑最可能失分**、但現在開始建立就能補救的項目：

### 高優先（立即開始）

1. **督導月報表**：從本月開始每月填寫，年底就有12份佐證
2. **個案電訪記錄**：月報內含電訪欄位，每月確認即可（B7-2）
3. **個案需求評估**：確認每位現役個案都有最近3個月內的評估記錄（B2-3）
4. **在職教育訓練積分**：登錄每位員工已參加的課程，確認年度20積分（B1-1）

### 中優先（本季完成）

5. **個別督導記錄**：督導每季對所轄照服員執行個別督導並記錄（A3-3）
6. **跨專業個案討論會**：本季辦理一次，邀請≥3種專業領域（B6-2）
7. **員工花名冊**：確認所有員工執業登錄狀態和督導比例（A6）

### 評鑑前（7月6日前）

8. 送達基本資料表及自評表（遲交即扣分）
9. 備齊至少3份現役個案服務契約（D1）
10. 確認業務負責人評鑑當日出席
