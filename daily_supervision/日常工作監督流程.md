# 日常工作監督流程

目的：建立可持續運作的日常監督流程，使機構平時執行紀錄能直接累積成評鑑佐證。

## 一、每日流程

1. 確認當日排班與派案紀錄是否完整。
2. 檢查是否有調班、代班、更換服務人員、暫停或復案。
3. 檢查是否有異常事件、申訴、服務衝突或主管機關通知。
4. 檢查新進或離職人員是否涉及 A5-A8 資料更新。
5. 將缺漏事項寫入「問題追蹤表」。
6. 將可歸檔資料寫入「佐證資料收件表」。

## 二、每週流程

1. 彙整本週每日檢查結果。
2. 檢查照服員每週工時與派案合理性。
3. 檢查督導紀錄、職前訓練、健檢、保險與缺失改善進度。
4. 產出本週缺口清單。
5. 更新進度追蹤與下週工作重點。

## 三、每月流程

1. 更新人員名冊、個案數與督導配置比例。
2. 檢查勞健保、勞退、保險、差勤與薪資佐證。
3. 更新 A項佐證文件索引。
4. 彙整可納入行政聯繫會議的服務品質與工作改善議題。

## 四、每季流程

1. 檢查行政聯繫會議是否完成。
2. 檢查年度業務計畫目標追蹤是否完成。
3. 檢查個別督導與團體督導紀錄。
4. 檢查工作手冊是否需修訂。
5. 彙整季度評鑑風險清單。

## 五、缺口處理流程

| 步驟 | 處理方式 |
|---|---|
| 發現缺口 | 寫入問題追蹤表。 |
| 指派負責人 | 依業務歸屬指定業務負責人、居督、行政或其他人員。 |
| 訂定期限 | 一般缺件7日內補齊，重大缺失依主管機關期限。 |
| 補件或改善 | 補上制度、紀錄、佐證、說明或改善措施。 |
| 追蹤結案 | 確認資料完整後標示結案。 |

## 六、資料歸檔流程

1. 新增資料先登錄於「佐證資料收件表」。
2. 判斷對應評鑑指標。
3. 放入對應指標資料夾或索引。
4. 若同一文件可支援多個指標，於索引中交叉註記。
5. 每月檢查索引是否完整。
