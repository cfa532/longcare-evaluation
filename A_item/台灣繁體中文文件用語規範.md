# 台灣繁體中文文件用語規範

本資料夾內所有評鑑文件、表單與後續增修內容，均採用台灣繁體中文與台灣長照評鑑常用語。

## 一、語言原則

- 使用台灣繁體中文，不使用簡體字。
- 優先採用附件評鑑基準之原文用語。
- 行政文件採正式、清楚、可稽核的語氣。
- 表單欄位以「可直接填寫、可供委員查閱」為原則。

## 二、固定用語

| 建議用語 | 避免用語 |
|---|---|
| 評鑑 | 评鉴、评价 |
| 長照 | 长照 |
| 長期照顧 | 长期照护、长期护理 |
| 照顧服務員 | 照服员、护理员 |
| 居家服務督導員 | 居督可作簡稱，但正式表單用全名 |
| 業務負責人 | 负责人 |
| 服務對象 | 个案、客户 |
| 家屬 | 家属 |
| 執行紀錄 | 执行记录 |
| 佐證資料 | 证明材料 |
| 查核 | 检查、稽查 |
| 差勤 | 考勤 |
| 勞健保 | 社保 |
| 勞退 | 退休金 |
| 職業安全衛生 | 职业安全 |
| 感染控制 | 感染管制亦可，但依A項文件統一用感染控制 |

## 三、表單撰寫規則

- 日期建議使用民國年或西元年，但同一份表單需一致。
- 涉及評鑑年度時，使用「115年度」。
- 人員欄位使用「員工姓名、職稱、到職日」。
- 個案相關欄位使用「服務對象」，避免使用商業化的「客戶」。
- 改善追蹤欄位至少包含「問題描述、原因分析、改善措施、負責人、完成期限、追蹤結果」。

## 四、後續文件命名建議

正式對外或評鑑備查文件可使用中文檔名，例如：

- A1_年度業務計畫書.md
- A2_工作手冊.md
- A3_行政聯繫會議紀錄.md
- A7_員工健檢與疫苗接種名冊.csv

目前資料夾內部分檔名保留英文，是為了電腦排序與後續批次處理方便；文件內容仍以台灣繁體中文為準。
