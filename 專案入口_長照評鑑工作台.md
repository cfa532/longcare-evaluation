# 長照評鑑工作台

固定入口：本對話由 Codex 協助操作 OpenClaw，進行長照機構評鑑資料之結構化管理、文件產出、表單維護與進度追蹤。

## 一、使用方式

使用者只需要在對話中直接交辦，例如：

- 幫我做 A1 年度業務計畫書正式版。
- 幫我把 A項表單轉成 Excel。
- 幫我檢查 A3 還缺哪些佐證。
- 幫我建立每季行政聯繫會議提醒。
- 幫我繼續做 B項。
- 幫我整理評鑑當天訪談稿。

Codex 負責：

1. 讀取評鑑基準與既有文件。
2. 依台灣繁體中文與長照評鑑用語產出文件。
3. 將成果放入本 OpenClaw 專案。
4. 維護索引、目錄、進度與下一步。
5. 必要時產出 Word、Excel、PDF 或其他備查格式。

## 三、固定工作規則

1. 所有對話與文件均使用台灣繁體中文。
2. 採用台灣長照與評鑑常用語。
3. 以「115年度居家式長期照顧服務機構評鑑基準」為主要依據。
4. 每項指標至少整理：
   - 制度文件
   - 執行紀錄
   - 追蹤改善
   - 訪談準備
   - 佐證文件索引
5. 文件需能供機構日常使用，也能在評鑑時直接備查。
6. 不刪除既有資料；若需搬移、歸檔或刪除，先確認再處理。
