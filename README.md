# 長照機構評鑑準備專案

語言版本：台灣繁體中文。  
評鑑依據：115年度居家式長期照顧服務機構評鑑基準。  
目前範圍：先建立 A項「經營管理效能」備查資料。

## 專案目標

本專案用於整理長照機構日常工作方案、制度文件、執行紀錄表單、追蹤改善紀錄與評鑑現場訪談準備資料，讓平時營運紀錄可直接轉為評鑑佐證。

## 目錄說明

| 目錄 | 說明 |
|---|---|
| `source/` | 原始評鑑基準文字檔，供校對條文使用。 |
| `A_item/` | A項經營管理效能之工作方案與表單模板。 |
| `A_item/templates/` | 可直接填寫或轉成 Word/Excel 的表單模板。 |
| `agents/` | Agents 日常監督角色、任務與回報規則。 |
| `daily_supervision/` | 每日、每週、每月、每季監督流程與紀錄表。 |

## 固定入口

| 文件 | 用途 |
|---|---|
| `專案入口_長照評鑑工作台.md` | 本專案的固定入口、使用方式與管理規則。 |

## 工作規則

1. 所有新增文件一律使用台灣繁體中文。
2. 優先採用評鑑基準原文用語。
3. 每項指標至少整理四類資料：制度文件、執行紀錄、追蹤改善、訪談準備。
4. 表單欄位需能直接供機構填寫與評鑑委員查閱。
5. 若要產出正式備查文件，可再轉為 `.docx` 或 `.xlsx`。
