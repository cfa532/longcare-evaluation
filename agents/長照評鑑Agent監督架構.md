# 長照評鑑 Agent 監督架構設計

**機構名稱**：專安誠健康事業有限公司附設高雄市私立專安誠居家長照機構  
**依據**：115年度居家式長期照顧服務機構評鑑基準  
**文件版本**：v2.0　更新日期：2026-06-13

---

## 一、設計原則

本架構以「平時運作即為評鑑準備」為核心，讓每位 Agent 的日常任務直接對應評鑑指標，確保：

1. 佐證文件隨時備查，無需臨時補件。
2. 缺失在平日即時發現、記錄、追蹤改善。
3. 年度評鑑時可由「評鑑總管 Agent」快速彙整所有佐證，產出自評表。

---

## 二、Agent 架構總覽

```
業務負責人
    │
    ├─ Agent A：個案照護督導員     （對應 B2/B3/B4/B5/B6/B7）
    ├─ Agent B：人員與訓練管理員   （對應 A5/A6/A7/A8/B1）
    ├─ Agent C：機構行政與品質稽核 （對應 A1/A3/A4/A9/D3/D4）
    └─ Agent D：評鑑準備總管       （整合所有 Agent 佐證，對應全部指標）
```

每個 Agent 均需：
- 有明確的負責人員
- 依排程自動提醒/執行任務
- 異常情況立即回報業務負責人
- 所有紀錄存入對應 Excel 工作簿

## 三、Agent A — 個案照護督導員

**對應指標**：B2（需求評估）、B3（照顧計畫）、B4（計畫追蹤）、B5（服務品質查核）、B6（跨專業轉介）、B7（開結案管理）

**負責人員**：居家服務督導員（曾怡萱）

### 3.1 日常任務

#### 每次服務後
- [ ] 確認照服員打卡紀錄是否完整（B5-1）
- [ ] 異常事件（未到班、服務中斷）立即記錄至《日常作業記錄簿》→「緊急異常事件報告」

#### 每月任務
| 任務 | 工具/表單 | 評鑑指標 |
|---|---|---|
| 對每位現役個案至少進行一次電訪 | 個案照護管理.xlsx → 月電訪季家訪紀錄 | B7-2 |
| 確認本月服務執行查核（至少1筆抽查） | 日常作業記錄簿 → 督導訪視紀錄 | B5-2 |
| 檢視各個案照顧計畫是否需要修正 | 個案照護管理.xlsx → 照顧計畫追蹤評值 | B4-1 |
| 確認本月收據開立完整 | 機構管理台帳.xlsx → 收費標準紀錄 | D2-2 |

#### 每季任務
| 任務 | 工具/表單 | 評鑑指標 |
|---|---|---|
| 對每位現役個案至少進行一次家訪 | 個案照護管理.xlsx → 月電訪季家訪紀錄 | B7-2 |
| 對每位個案重新執行需求評估（或確認無變化） | 個案照護管理.xlsx → 需求評估紀錄 | B2-3 |
| 辦理跨專業個案討論會（≥3種專業領域，含轉介情境） | 個案照護管理.xlsx → 跨專業轉介個案討論 | B6-2 |
| 彙整跨專業轉介追蹤結果 | 個案照護管理.xlsx → 跨專業轉介個案討論 | B6-1 |

#### 新個案開案時（7個工作天內）
- [ ] 完成需求評估（B2-1）→ 存入「需求評估紀錄」
- [ ] 依評估結果擬定照顧計畫（B3-1）→ 存入「照顧計畫書」
- [ ] 確認計畫與評估結果一致（B3-2）
- [ ] 向個案/家屬說明計畫內容並取得確認（B7-1）
- [ ] 執行初訪 → 存入「督導訪視紀錄」（類別勾選□初訪）
- [ ] 記錄至「個案開結案管理」

### 3.2 回報規則

- **立即回報**：個案發生緊急/意外事件（B8）
- **當日回報**：照服員無故未到班、個案強烈申訴
- **每月回報**：月電訪/季家訪完成率、照顧計畫修正清單
- **每季回報**：個案討論會辦理情形、轉介追蹤結果

## 四、Agent B — 人員與訓練管理員

**對應指標**：A5（員工權益）、A6（人力設置）、A7（健康檢查）、A8（職前訓練）、B1（專業知能）

**負責人員**：業務負責人 / 行政人員

### 4.1 日常任務

#### 新進人員到職時
- [ ] 建立員工花名冊記錄（人員管理台帳.xlsx → 員工花名冊）A6
- [ ] 確認勞/健保及勞退提繳完成（A5-3）
- [ ] 確認到職前健康檢查完成（胸部X光、糞便、血液、尿液、B肝）A7-1
- [ ] 確認執業登錄或報備完成（A6-1）
- [ ] 啟動職前訓練計畫，1個月內完成16小時訓練（A8-1）
- [ ] 記錄訓練紀錄至「職前訓練記錄」工作表

#### 每月任務
| 任務 | 工具/表單 | 評鑑指標 |
|---|---|---|
| 確認本月在職教育訓練課程辦理情形 | 人員管理台帳.xlsx → 在職教育訓練 | B1-2 |
| 更新員工每週工時統計 | 機構管理台帳.xlsx → 排班派案記錄 | A9 |

#### 每年任務
| 任務 | 工具/表單 | 評鑑指標 |
|---|---|---|
| 辦理全員年度健康檢查（胸部X光、血液、尿液） | 人員管理台帳.xlsx → 健康檢查記錄 | A7-2 |
| 彙整每位員工年度長照積分（確認每人≥20積分） | 人員管理台帳.xlsx → 在職教育訓練（積分統計區） | B1-1 |
| 確認感控≥4積分、原民文化≥1積分、多元文化≥1積分 | 同上 | B1-1 |
| 配合流感疫苗政策造冊施打，未施打者記錄原因 | 人員管理台帳.xlsx → 疫苗接種記錄 | A7-4 |
| 審閱工作手冊並完成年度修訂（A2-3） | 工作手冊（另存） | A2 |
| 確認工作手冊員工閱讀簽認完整（A2-4） | A_item/templates/A2_staff_manual_reading_signoff.csv | A2 |

### 4.2 督導比例自動警示

人員管理台帳 → 員工花名冊 內建公式：
- 督導人數：`=COUNTIFS(C:C,"督導",F:F,"")`
- 個案數（手動更新）
- 比例警示：若每位督導服務個案 > 60 人，自動顯示「✗ 不符合（請增加督導）」（A6-2）

### 4.3 回報規則

- **立即回報**：員工未投保勞/健保、督導人力不足（A6-2 觸發警示）
- **每月回報**：訓練課程辦理情形、新進員工職前訓練進度
- **每年回報**：全員積分達標情形、健康檢查完成狀況

### 6.2 自評表評分估算公式

評鑑計分（供自評使用）：
- **A項**：9個指標 × 4分 = 最高 36分，佔 43%
- **B項**：8個指標 × 4分 = 最高 32分，佔 38%
- **D項**：4個指標 × 4分 = 最高 16分，佔 19%
- **加分項**：最高 +5分（加分後以佔比計算）
- **扣分項**：每案扣總分 0.5分，上限扣 2分

### 6.3 年度評鑑準備時程

| 時間點 | 任務 | 負責 Agent |
|---|---|---|
| 評鑑前12個月 | 開始系統性維護所有記錄 | A/B/C |
| 評鑑前6個月 | 確認A項所有佐證文件完整性 | Agent D |
| 評鑑前3個月 | 啟動B項佐證文件全面清查 | Agent D |
| 評鑑前2個月 | 完成自評表填寫，識別可能失分項 | Agent D |
| 評鑑前1個月 | 補強弱項，準備訪談口徑 | 業務負責人 |
| 評鑑前2週 | 整備現場查閱文件（分類裝訂） | 全體 |
| 評鑑截止日 115年7月6日 | 送達基本資料表及自評表（遲交扣分） | Agent D |

## 七、Agent 任務日曆

### 每日（工作日）
- Agent A：確認照服員到班打卡紀錄 → 異常立即記錄
- Agent C：確認當日有無新增申訴/意見反應

### 每月（月底前）
- Agent A：完成全部個案電訪 + 更新月電訪季家訪紀錄
- Agent B：更新訓練出席紀錄 + 排班工時統計
- Agent C：處理本月申訴 + 更新排班派案記錄

### 每季（1月、4月、7月、10月底前）
- Agent A：完成全部個案家訪 + 需求重新評估 + 個案討論會
- Agent C：行政聯繫會議 + 個別督導 + 業務計畫季度檢視

### 每年（12月底前）
- Agent B：全員健康檢查 + 積分達標確認 + 疫苗施打
- Agent C：次年度業務計畫（含董監事會議）+ 滿意度調查
- Agent D：年度佐證文件整理歸檔 + 自評表初稿
